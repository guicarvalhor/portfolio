Resumo Profissional

Desenvolvedor Front-End e Design UX/UI. Atualmente trabalho no IPEDF em Brasília-DF, utilizando HTML, CSS, Angular, Git e Figma para prover experiências funcionais e intuitivas.

Atuei na criação de telas para o projeto Sistema de Avaliação de Trabalhos (SAT)/Prêmio IPEDF, participei do redesign do Questionários IPEDF (LIMESURVEY), pude trabalhar tanto no redesign como na implementação do código do Catálogo de Mapas (GEONODE) e do Anuário Estatístico do DF.

Criei componentes com alta fidelidade para a biblioteca de componentes do Storybook do IPEDF, utilizando Sass e seguindo os padrões criados no Sistema de Design.

Participei ativamente na criação do Sistema de Design, usando Figma para criar componentes e padrões visuais que promovem consistência, eficiência e acessibilidade em todos os produtos e serviços, bem como, validei layouts e criei fluxos de usuários no Figma.

Tenho histórico comprovado de colaboração em equipes: atuando como designer e no desenvolvimento, seguindo boas práticas de código e garantindo a qualidade e usabilidade das interfaces de usuário.

Consigo combinar meu amor por tecnologia com minha paixão por design. Gosto de usar minha criatividade e habilidades de resolução de problemas para criar experiências digitais que não só tenham uma ótima aparência, mas também funcionem perfeitamente para o usuário. Acredito que o aprendizado contínuo é a chave para o sucesso, e estou sempre em busca de novos desafios para me manter motivado e engajado.
